# ⚖️ Anlaşmazlık Çözücü

### Aynı anda iki farklı şeyi mi istediniz? Bırakın jüri karar versin!
İki seçenek arasında (Örn: Hangi filmi izlesek? Akşama ne yesek? Nereye gitsek?) kararsız kaldığınızda, size tarafsız (!) ve oldukça **absürt gerekçelerle** karar veren eğlenceli ve modern bir web uygulamasıdır.

## 🚀 Özellikler
- **Eğlenceli Gerekçeler:** Kazanan seçeneği belirlerken yapay, eğlenceli ve komik kurgusal bahaneler üretir.
- **Modern ve Dinamik Arayüz:** Göz yormayan, premium renk geçişleri, modern tasarımlar (glassmorphism) ve animasyonlarla süslüdür.
- **PWA (Progressive Web App):** Sadece tarayıcı üzerinden web sitesi olarak değil, mobil cihazınıza doğrudan indirip **gerçek bir uygulama gibi** kurabilirsiniz!

## 📲 Mobil Uygulama Olarak Kullanım
Bu uygulama kurulum gerektiren (PWA) modern standartlara sahiptir. Telefonunuza indirmek için:
1. Sitenin yayındaki GitHub Pages bağlantısını telefon tarayıcınızda açın.
2. Tarayıcı menüsünden **"Ana Ekrana Ekle" (Add to Home screen)** seçeneğine dokunun.
3. Uygulamanız telefonunuzda belirsin! (Dilerseniz PWABuilder üzerinden APK'ya da çevrilebilir).

## 🛠 Kullanılan Teknolojiler
- Vanilla JavaScript
- Modern HTML5 & CSS3
- Service Worker & Web Manifest (PWA için)

*Bu proje, günlük hayattaki küçük kararsızlıkları mizahi bir şekilde çözmek amacıyla geliştirilmiştir.*
